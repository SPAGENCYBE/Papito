FIREBASE INSTELLEN VOOR ECHTE ONLINE OPSLAG

Deze website is technisch voorbereid voor:
1. online condoleanceregister
2. foto-upload
3. goedkeuren/weigeren in admin
4. kaarsjesteller

Stappen:
1. Ga naar https://console.firebase.google.com/
2. Maak een project, bijvoorbeeld Papito.
3. Maak een Web App aan.
4. Kopieer de firebaseConfig.
5. Open index.html en fotos.html.
6. Zoek:
   const USE_FIREBASE = false;
   Maak daarvan:
   const USE_FIREBASE = true;
7. Vul firebaseConfig in.
8. Activeer Firestore Database.
9. Activeer Storage.
10. Publiceer opnieuw naar GitHub Pages.

Collecties:
- memories
- candles

Let op:
Voor echte beveiliging moet je Firebase Rules correct instellen.
